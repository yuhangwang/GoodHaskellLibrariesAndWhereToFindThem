# GoodHaskellLibrariesAndWhereToFindThem

## Emededable Haskell Interpreter
* [hint](https://github.com/mvdan/hint)
  - Features:
    - load Haskell modules
    - type-check and evaluate strings and convert them to values
    - type-safe
    - thread-safe
